# Week 3 — Prompt Examples  
**Focus:** Content Creation, Productivity, and Prompt Design

This file documents reusable prompts used to generate professional, portfolio-ready content using ChatGPT. Each prompt is designed with clear role definition, audience targeting, and output constraints.

---

## 1️⃣ Blog Topic Research Prompt

**Goal:** Generate timely, audience-relevant blog ideas.

**Prompt:**
>You are an AI content strategist. Generate 10 blog topics for college graduates seeking immediate employment in technology. Each topic should be timely, practical, and optimized for SEO. Include a short rationale for each topic.

**Why it works:**
- Defines a clear role (content strategist)
- Specifies the target audience
- Requests structured, actionable output

---

## 2️⃣ Professional Blog Outline Prompt

**Goal:** Create a structured blog outline quickly.

**Prompt:**
>Create a clear blog outline (H1–H3) for a post titled “AI Skills College Graduates Need to Get Hired Faster.” Include an engaging hook, 4–5 main sections, and a strong call to action.

**Why it works:**
- Enforces structure (H1–H3)
- Keeps content focused and scannable
- Optimized for professional publishing

---

## 3️⃣ Tone-Controlled Rewrite Prompt

**Goal:** Adapt content for different audiences.

**Prompt:**
>Rewrite the following paragraph for a professional but encouraging tone aimed at recent graduates. Keep it concise and action-oriented.

**Why it works:**
- Explicit tone control
- Audience-specific language
- Produces consistent professional voice
